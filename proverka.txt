ОСНОВНЫЕ КОМАНДЫ git

git init 
создание репозитория в текущей папке

git add
добавление файла в репозиторий

git commit -m file_name.txt "комментарий"
фиксация текущих изменений с комментарием

git status
текущий статус git

git log
список всех коммитов

git checkout HASH
HASH - хэш, первые 6 символов предыдущих коммитов, 
вернуться в состояние предыдущих коммитов

git branch 
список всех веток

git branch new_vetka
создание новой ветки

git checkout -b new_vetka2
создание ветки и переключение на нее

git checkout main
git checkout new_vetka2
переключение на ветку main / new_vetka2

git revert HEAD
отмена последнего изменения

git merge new_vetka
слияние текущей ветки с new_vetka

git branch -d new_vetka
удаление ветки new_vetka

git clone https://github.com/magix-g/bj.git
клонирование удаленного репозитория на локальный компьютер

ПОРЯДОК РАБОТЫ С УДАЛЕННЫМ РЕПОЗИТОРИЕМ
========================================

git remote add origin1 https://github.com/magix-g/bj.git
Подключиться к удаленному репозиторию и присвоить имя соединению origin

git branch -M main
создать ветку main

git push -u origin main
выгрузить локальный файл на удаленный репозиторий через push
origin - название соединения
main - название ветки

git push
выгрузить локальный файл к уже ранее присоединенному репозиторию

git pull
скачать измененный файл с удаленного репозитория
